# Hybrid Vulnerability Index (Education Under Threat)

This document provides documentation for the vulnerability indices generated by the `scripts/calculate_hybrid_vulnerability.py` analysis pipeline.

## Methodology

The index is a hybrid composite score designed to estimate educational risk in conflict-affected regions where granular education data is sparse. It combines a **National Education Baseline** (based on comprehensive historical education data) with a **Granular Conflict Proxy** (derived from localized ACLED conflict events).

### Calculation Logic
The index is calculated as a weighted composite score:
`Vulnerability Index = (National Baseline * 0.6) + (Conflict Proxy * 0.4)`

1.  **National Baseline**: A normalized [0, 1] index derived from annual national-level indicators (e.g., enrolment, completion, out-of-school rates). It captures the overall deterioration or improvement of the national education system.
2.  **Conflict Proxy**: A normalized [0, 1] index derived from the total number of conflict events recorded at the Admin2 (district) level. This serves as a proxy for physical access barriers and systemic disruption in a specific region.

## Data Dictionary

| Column | Type | Description |
| :--- | :--- | :--- |
| `Admin2` | String | The administrative district or province. |
| `year` | Integer | The calendar year of the record. |
| `baseline` | Float [0, 1] | National educational vulnerability score for that year. |
| `Events` | Integer | Total ACLED conflict events in this Admin2 region that year. |
| `conflict_proxy` | Float [0, 1] | Normalized event intensity for this region (relative to the highest conflict year in the dataset). |
| `vulnerability_index` | Float [0, 1] | The final risk estimate. **Higher scores indicate higher vulnerability.** |

## Interpretation
*   **0.0 - 0.3**: Low educational vulnerability; stable access and low conflict intensity.
*   **0.3 - 0.6**: Moderate vulnerability; emerging conflict pressures or systemic education strain.
*   **0.6 - 1.0**: High vulnerability; significant conflict disruption likely compromising educational access and continuity.

*Note: This index is an estimation model. It should be used to support decision-making, not as a standalone source of truth for field operations.*
